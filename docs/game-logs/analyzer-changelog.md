# Log Analyzer Changelog

日志复盘与分析工具的变更记录。

---

## 2026-05-14

### 新增
- 新增 `tools/export_game_log_html.py`，可将 `game_log_*.json` 导出为单文件 HTML 复盘报告。
- 支持每局独立切换、每墩围桌展示、胜者/先手标记、牌型/牌域/排序值明细。
- 支持每局初始手牌、每墩出牌前/出牌后手牌，并高亮本墩实际出牌。
- 支持庄家顺延原因展示：无可定主牌、玩家选择不定主、AI 放弃定主。
- 支持人工订正备注，并可导出订正 JSON。
- 新增 `docs/game-logs/standard_log.json` 作为标准样例日志。

### 检测项
- 每墩出牌顺序、先手延续、赢家复算、牌点复算。
- 攻方得分连续性与结算字段复算。
- 同域跟牌、对子必须跟对子。
- 庄家顺延、实际庄家队伍等级与本局打级一致性。

### 兼容
- 对旧日志缺失 `hands_after` 的情况，HTML 导出器会用 `hands_before - plays` 推导出牌后手牌。
- 对进行中的未结算局，跳过空 `settlement` 字段复核，避免误报。
